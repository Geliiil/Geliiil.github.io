# Geliiil網頁基本架構
{{< figure src="./computer.png" width="600" alt="設計"  class="center" >}}
本網頁以Hugo架構所撰寫，並部署在GitHub。
網頁本身架構我分成以下5個部分：
1. Home (主頁)
2. Posts (發文頁面)
3. About (自我介紹)
4. Collections (我的音樂收藏)
5. Tags (基本分類)

第4個部分是因為我平常有在收集一些專輯，想說專輯收了一些了想要找個（好？）地方分享，所以特別建立了Collections來放我的專輯收藏。但因為平常比較忙碌，內容可能更新會有點慢～

# 用 Hugo 架構架設網頁之心酸史
Hugo 其實有很多模板（theme）可以直接下載套用。整體來說，Hugo網頁做起來其實非常的快，只要把內容填入即可！而我第一版網站就是用theme套一套就用出來了（你現在看到的是第二版），只花了一個上午就用好了，非常快速！

雖然很快就做完了，但總覺得有些不滿意，因此開始有了想建立自己的模板的念頭...

所以我就Google了一下（這個時代還不是先用AI查，非常老人的我👵）theme 要怎麼建立。結果官方文件寫的太簡易了，害我以為這個架構很簡單，又加上以前寫過幾次 React，會一點點 HTML、CSS、JavaScript，想說應該難不倒我。最後我整整搞了3天才把網站做起來。

一路上遇到很多問題，譬如 Hugo 整體架構我就研究了好久，還有 Hugo 本身有自己的語法，沒學過根本看不懂，再來不知道為什麼很多時候 CSS 渲染不上去...。但關關難過關關過，最後總是走過來了，在這3天內學到很多，也發現我沒事找事做...所以建議如果有時間再自己做theme就好，不然直接套現成的其實就很不錯了，而且好像可以自己寫你像要的theme覆蓋原本的。
